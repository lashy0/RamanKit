# RamanKit

[![CI](https://github.com/lashy0/RamanKit/actions/workflows/ci.yml/badge.svg)](https://github.com/lashy0/RamanKit/actions/workflows/ci.yml)

RamanKit is a Python library for working with Raman spectra, collections of spectra, and hyperspectral Raman images through a typed, scientifically explicit core API.
